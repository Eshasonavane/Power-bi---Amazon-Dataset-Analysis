# Amazon Sales Analysis Dashboard

## Description
This Power BI dashboard analyzes Amazon sales data to visualize trends, performance metrics, and customer insights. It helps understand product performance, revenue patterns, and customer behavior.

## Features
- Interactive dashboards for exploring sales data
- Visualizations of top-selling products, revenue, and sales trends
- Insights into customer demographics and purchase patterns
- Easy-to-use Power BI interface with filters and slicers

## Dataset
The dataset contains Amazon sales records, including product category,sub-category,Region, Total sales, order date, customer id, products quantity,unit,Discount and profit. 

## How to Use
1. Download the `.pbix` Power BI file from this repository.
2. Open it in **Power BI Desktop**.
3. Explore the dashboards and interact with filters and slicers to analyze the data.
